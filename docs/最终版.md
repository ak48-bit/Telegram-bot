## 三、最终版说明

当前最终版已包含以下能力：

1. Admin 按钮创建 Agent
2. Agent 按钮创建 Promoter
3. Promoter 绑定后显示 Telegram Username 和 Telegram ID
4. Player 进入后显示 Telegram Username 和 Telegram ID
5. Player 按钮提交 Game ID
6. Admin / Agent / Promoter 可查看 Player 数据
7. p_B01 / p_C001 玩家入口正常
8. Existing Player 再点链接不会覆盖归属
9. 一个 Telegram 账号只能绑定一个身份
10. 角色互斥已修复

---

## 四、最终版操作入口

Admin：

```text
/admin → Add Agent → 输入 Agent Code → 输入 Agent Name → Confirm
```

Agent：

```text
绑定成功 → Add Promoter → 输入 Promoter Code / Name / Affiliate Link → Confirm
```

Promoter：

```text
绑定成功 → 获取 p_B01 Bot Share Link → Share 给 Player
```

Player：

```text
点击 p_B01 / p_C001 → Submit Game ID → 提交成功
```
