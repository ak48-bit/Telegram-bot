# 线上地推组 - 量化指标体系

## 一、每人每天必看指标（5个核心数字）

| 指标 | 英文 | 怎么算 | 红线 |
|------|------|--------|------|
| 首存人数 | FTD | 每天新增首存玩家数 | <5人/天要预警 |
| 存款总额 | DPS | 当天总存款金额 | 周环比掉30%要预警 |
| 提款总额 | WDR | 当天总提款金额 | >存款70%要查原因 |
| 净流水差 | DIFF | DPS-WDR | 不能为负数 |
| 日均首存 | AVG FTD | 本月FTD÷工作天数 | <10要谈话 |

## 二、每组每周必看指标

| 指标 | 怎么算 | 用途 |
|------|--------|------|
| 周FTD增长率 | (本周-上周)÷上周 | 看趋势 |
| 人效比 | 组总DIFF÷组人数 | 排名用 |
| 零单率 | 零单天数÷总工作天数 | 低于10%正常 |
| 玩家留存率 | 7天后还在玩的÷总首存 | >30%算合格 |

## 三、每月必出报表

1. **个人月度对比表** - 你现在已经在做（线上办公业务员数据对比）
2. **组别月度排名** - 按DIFF排名
3. **平台维度分析** - PH30 vs PH09 对比
4. **异常人员清单** - 连续2个月下降的人
